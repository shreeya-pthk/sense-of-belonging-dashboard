# 📊 Sense of Belonging Survey Analytics Dashboard

## 🏆 Award
Recognized with the Student Personnel Area 
Creative Innovator Award — Kent State University (2026)

## 📌 Overview
An end-to-end Power BI analytics platform built to help 
KSU leadership make data-driven decisions on student 
success programs.

## 🎯 Problem Statement
University leadership had no centralized way to track 
student engagement trends across 4 years of survey data.

## 💡 Solution
Built an interactive drill-through dashboard ingesting 
6,000+ records from 4 siloed systems into one 
executive-ready platform.

## 📈 Key Results
- ✅ Reduced manual data prep time by 40%
- ✅ Identified 3 critical engagement gaps
- ✅ Delivered within 4 weeks of onboarding
- ✅ Directly influenced resource allocation strategy

## 🛠️ Tools Used
Power BI | DAX | Power Query | Excel | ETL Pipelines
